The Machine: a live forensic pass

Four fictional outlets cover the same fictional event, each with its own framing. Run the pass to watch neutralization, claim extraction, and consensus-threshold scoring happen for real — then drag the threshold and watch claims cross it.

How this demo differs from the production pipeline

In production, framing neutralization and claim/graph extraction run via open-source LLMs served through Fireworks AI API on AMD Instinct hardware (MI325X/MI355X), with sentence-transformer embeddings running on an AMD Radeon GPU pod via ROCm for the clustering step. Specific model choices are pending benchmarking once API access is confirmed — leading candidates are DeepSeek-V4-Pro and Llama 3.3 70B for extraction and synthesis. This browser prototype calls Claude directly to test the interaction choreography — the prompts, the staging, the threshold math — before infrastructure is wired in. The prompts below are simplified analogues of the Forensic Extraction Agent (neutralization + claim extraction) and Consensus Alignment Agent (cross-source clustering); in production the clustering step uses sentence-transformer + DBSCAN on GPU rather than a single LLM call.
