Source Profile — Vertical Picker

Decision 1 of 3

Each source has 3 independent reputation profiles — one per vertical (GEOPOLITICS, ECONOMICS, TECHNOLOGY). The question: how does the user switch between them?

A. Vertical pills — active tab selected

Like Sources page ArchetypePills. User clicks a tab, the radar + stats + sparklines re-render for that vertical. "All" not meaningful here — each vertical is independent.

Matches existing ArchetypePills UX pattern
Can show different archetypes per vertical (a source may be EB in GEOPOLITICS but NF in ECONOMICS)
Requires user to click to see other verticals — hidden by default
Vertical context changes when switching — user must remember what they saw

B. All 3 verticals shown simultaneously (stacked)

Three rows, one per vertical. Each has its own radar + stats. User scrolls or tabs between sections. "At a glance" comparison.

Event markers from different verticals interleaved on timeline

C. Single default vertical — no picker

Pick one vertical as the default view. The page shows only that vertical's data. Simplest implementation, but the user can't see the others without going somewhere else (e.g. a settings dropdown).
